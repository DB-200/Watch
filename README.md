# Diver Watch (WPF)

A polished WPF clock app styled like a premium diver's watch with smooth hand animation and high‑quality vector graphics.

## Run

```cmd
cd /d "C:\Users\Dev\Documents\New project\Watch"
dotnet run
```

## Build

```cmd
cd /d "C:\Users\Dev\Documents\New project\Watch"
dotnet build
```

## Notes

- The watch face is fully vector drawn in XAML.
- Hands are animated in real time using `CompositionTarget.Rendering` for smooth motion.

## Project Structure

- `Watch/` WPF application
- `.gitignore` standard build and IDE outputs
